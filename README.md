# munkun.github.io

Root user site for Muniverse Studio.

## Purpose

Hosts `app-ads.txt` at the root of `https://munkun.github.io/` as required by the IAB Tech Lab standard for AdMob developer verification.

## Files

- `app-ads.txt` — AdMob ads.txt authorization entry
- `index.html` — Landing page linking to privacy policy
